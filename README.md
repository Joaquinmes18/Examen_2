# TechSolutions - Catálogo Interactivo de Servicios Digitales

**Joaquin Elias**
**Enfoque:** Arquitectura SASS, Maquetación Estructural y Diseño Responsivo.
- Construcción de la estructura HTML5 aplicando etiquetas semánticas (`<header>`, `<main>`, `<section>`, `<article>`).
- Configuración global de SASS, definiendo la paleta de colores, tipografías (variables) y mixins reutilizables.
- Implementación del sistema de grillas (CSS Grid y Flexbox) para garantizar la adaptabilidad del layout en 3 breakpoints (Móvil, Tablet y Desktop).
- Estilización de la sección *Hero* y el layout del catálogo de tarjetas de servicios.
 **Cristhian Claros**
**Enfoque:** Lógica JavaScript, Interactividad del DOM y Validaciones.
- Desarrollo de la interactividad del menú de navegación (Menú hamburguesa / Toggle) para la vista móvil.
- Implementación del sistema de filtrado dinámico de servicios por categoría y la actualización del contador en tiempo real usando JavaScript.
- Lógica de validación del formulario de contacto (prevención de envío por defecto, control de campos vacíos y uso de Expresiones Regulares para el correo electrónico).
- Maquetación y estilización SASS de la sección de contacto, incluyendo los estados visuales de error y éxito de los inputs.
